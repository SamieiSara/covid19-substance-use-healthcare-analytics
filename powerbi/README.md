# Power BI Dashboard

## Objective

This dashboard explores changes in substance-related harms during the COVID-19 pandemic using CIHI healthcare data.

## Key Metrics

- Emergency Department (ED) visits
- Hospitalizations
- Substance-related deaths
- Opioid-related harms
- Demographic trends
- Income-based disparities

## Visualizations

- KPI Cards
- Monthly Trends
- Substance Comparison Analysis
- Age Distribution Analysis
- Income Quintile Analysis
- Death Indicators

## Tools

- Power BI
- Power Query
- DAX
